# Food Delivery Frontend

Next.js frontend for the food delivery application with TypeScript and TailwindCSS.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Production

```bash
npm start
```

## Testing

```bash
npm test
```

## Components

- `Menu` - Displays food items from the menu
- `Cart` - Manages cart items and quantities
- `Checkout` - Form for delivery details
- `OrderStatus` - Real-time order tracking

## Features

- Next.js 14 with App Router
- TypeScript for type safety
- TailwindCSS for styling
- Real-time order status updates
- Responsive design
